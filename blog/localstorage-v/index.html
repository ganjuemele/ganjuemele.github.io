数据库版本更新

关于数据库版本更新的解决方法

March 20, 2020 么了

1 分钟阅读

  • Twitter

  • Facebook

  • LinkedIn

  • Email

window.localStorage.setItem('version','1.0.0')

初始的 Version ‘1.0.0’, 更改为 ‘2.0.0’，在 localStrage 中的数据需要新加入个 Date

const version = window.localStorage.getItem('version') || '0'；
const recordList: Record[] = JSON.parse(window.localStorage.getItem('recordList') || '[]');

if(version < '2.0.0'){
	//数据库升级
	//遍历之前的数据给添加日期，使数据结构相同
	recordList.forEach(record => {
		record.createdAt = new Date(0)
	})
	//保存数据
	window.localStorage.setItem('recordList',JSON.stringify(recordList))
}
//设置新版本号
window.localStorage.setItem('version','2.0.0')

版本更新前先获取之前的版本号
