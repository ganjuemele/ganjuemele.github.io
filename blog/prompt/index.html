prompt的确定和取消事件

March 22, 2020 么了

1 分钟阅读

  • Twitter

  • Facebook

  • LinkedIn

  • Email

prompt事件触发，未输入标签名点击取消，也会新增一个空白标签， 这是因为点击取消， name === null

const name = window.prompt('请输入标签名');
if (name === null){
	// console.log(name,'null')
}else if (name === '') {
	window.alert('标签名不能为空')
} else (this.dataSource) {
	this.$emit('update:dataSource', [...this.dataSource, name])
}

⚠️这里要把（name===null）的情况放在较前位置进行判断

优化下

if (name === '') {window.alert('标签名不能为空')}
else if (name !== null && this.dataSource) {
	this.$emit('update:dataSource', [...this.dataSource, name])
}
