模板字符串（template string）是增强版的字符串，用反引号（`）标识

它可以当作普通字符串使用，也可以用来定义多行字符串，或者在字符串中嵌入变量。

<navigator class="vbox" open-type="navigate"
		a:for="{{ruleList}}" a:key="{{item.index}}"
        url="{{item.actType==='C'
		?`/pages/mine/cashCouponInfo/cashCouponInfo?info=${item.info}`
		:`/pages/couponOperate/couponOperate?info=${item.info}`}}">
</navigator>

反引号中可以写一个字符串，并且插入使用 ${变量名} 。
